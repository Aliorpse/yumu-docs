这个功能可以从机器人的数据库中获取玩家 7 天之内表现较好的成绩（包含坟场谱面或者其他不含榜单谱面上的成绩）。

:::: info 使用方法

!recentbest / !rb / 最近优秀成绩 (`:` ::gamepad:: 游戏模式) ( ::address-card:: 玩家名)

::: tabs

@tab ::gamepad:: 游戏模式

可选。参见：[成绩/绑定游戏模式 (SM)](./mode.md)

@tab:active ::address-card:: 玩家名

可选。参见：[字段/玩家名](../../tips/field/player.md)

:::
