这个功能可以分析最好成绩榜，抽出不完美 (choke) 成绩，并预测当这些成绩被修补成全连 (full combo) 后，能（理论上）获得多少表现分。

:::: info 使用方法

!bestfix / !bf / 修补最好成绩 (`:` ::gamepad:: 游戏模式) ( ::address-card:: 玩家名)

::: tabs

@tab ::gamepad:: 游戏模式

可选。参见：[成绩/绑定游戏模式 (SM)](./mode.md)

@tab:active ::address-card:: 玩家名

可选。参见：[字段/玩家名](../../tips/field/player.md)

:::
