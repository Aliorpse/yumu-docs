---
title: 绑定游戏模式
icon: list
order: 1
---

用户在绑定后，一般都需要修改自己的绑定游戏模式，来让机器人确定发送成绩等情况下，应该去哪个游戏模式里寻找。

如果用户使用的部分功能，需要输入游戏模式，会优先使用群聊绑定的游戏模式，其次是用户绑定游戏模式。参见：[成绩/绑定群聊模式 (GM)](./groupmode.md)

::::: info 使用方法

!mode / sm / mo / 模式 (`:` ::gamepad:: 游戏模式)

:::: tabs

@tab ::gamepad:: 游戏模式

可选。`:` 可省略。

默认移除玩家绑定的游戏模式。此时机器人在需要指定游戏模式时，只会使用玩家主页的默认游戏模式。

::: details 游戏模式对应表

| 模式 | 简称 | 数字 | 名称 |
| :-: | :-: | :-: | :-: |
| ::circle-dot:: 主模式 | o | 0 | osu、std、standard |
| ::drum:: 太鼓模式 | t | 1 | taiko |
| ::apple-whole:: 接水果模式 | c | 2 | catch、fruits |
| ::keyboard:: 下落模式 | m | 3 | mania |

私服指定的游戏模式、特殊的键位模式如下：

| 模式 | 简称 | 数字 | 名称 | 备注 |
| :-: | :-: | :-: | :-: | --- |
| ::circle-dot:: 主模式 RELAX | or、rx0 | 4 | osurelax、stdrelax、osurx | 偏偏要上班服务器特有 |
| ::drum:: 太鼓模式 RELAX | tr、rx1 | 5 | taikorelax、taikorx | 偏偏要上班服务器特有 |
| ::apple-whole:: 接水果模式 RELAX | cr、rx2 | 6 | catchrelax、catchrx | 偏偏要上班服务器特有 |
| ::keyboard:: 下落模式 7K | 7k | 7 | mania7k | 仅排位模式中区分 4/7K 使用 |
| ::circle-dot:: 主模式 AUTOPILOT | oa、ap | 8 | osuautopilot、stdap、osuap | 偏偏要上班服务器特有 |

:::

::::

:::::

::: tip 提示

在这里绑定的 ::gamepad:: 游戏模式，可以与玩家主页的默认游戏模式，或者玩家在其他机器人内绑定的游戏模式不同。

这在同时使用多种机器人查询成绩时尤其有用。比如仅在本机器人下查询接水果模式成绩，在其他机器人下查询主模式成绩等。

但是，这个绑定的游戏模式会被群聊游戏模式所==覆盖==（如果有）。

如果在部分==限定玩家讨论游戏模式==的群聊中，您发现自己的绑定模式未生效，则可能是因为这个群聊已经设定了群聊游戏模式，覆盖了您的绑定模式。

:::
