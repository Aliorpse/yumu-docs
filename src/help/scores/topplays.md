这个功能可以查询全服表现分最高的成绩。

:::: info 使用方法

!topplays / !tp / 顶流成绩 (`:` ::gamepad:: 游戏模式) (`#` ::file:: 页码)

::: tabs

@tab:active ::gamepad:: 游戏模式

可选。参见：[成绩/绑定游戏模式 (SM)](./mode.md)

默认查询 ::circle-dot:: 主模式。

@tab ::file:: 页码

可选。`#` 可省略。默认第 1 页。

一页 50 个结果，最大可跳转到 20 页。

:::
