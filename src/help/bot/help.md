您可以通过 QQ 群聊，查询机器人基础指令，以及以下所有模块的指令。

但是您既然已经看到这里了，不如就直接在文档网站里查吧。

::::: info 使用方法

!help / h / 帮助 ( ::table-list:: 模块)

:::: tabs

@tab ::table-list:: 功能分类

可选。需要查询的功能名称，或是功能所属的模块名（一组功能）。

::: details 模块对应表

| 模块 | 简称 | 名称 |
| :-: | :-: | --- |
| bot | o | 内部、机器 |
| scores | s | 成绩、分数 |
| player | p | 玩家、用户 |
| map | b | 谱面、图 |
| tournament | m | 比赛、房间 |
| chat | c | 聊天 |
| fun | f | 娱乐 |
| maimai | i | 舞萌、中二 |
| aid | a | 辅助 |
| custom | u | 自定 |
| private | e | 私服 |

:::

::::

:::::

::: tip 提示

用 !help、或是用 !帮助 调用出来的帮助页面，是简化版。它只覆盖了大多数用户可能会使用的功能，方便您快速上手。

如果想获取所有指令，必须使用 !h。

:::
