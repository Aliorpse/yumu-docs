超级管理员可以使用的一个简单的机器人回声功能。

:::: important 超管专属使用方法

!echo / ec / 回声 ( ::phone:: 群组号/玩家号) ( ::tv:: 内容)

::: tabs

@tab:active ::phone:: 群组/玩家号

可选。如果不填写，则直接在当前聊天中回应。

- 群组号：
    - 如果输入数字，默认为群组号。
    - 输入格式也可以为 `group=`114514。
- 玩家号：
    - 输入格式必须为 `qq=`114514。
    - 对于回声传递给特定用户（私聊），虽然功能上没有问题，但是很容易导致机器人账号被意外封禁。即使你是超级管理员，也最好不要这么做。

@tab ::tv:: 内容

需要回应的内容。

目前仅文本消息经过了测试，其他类型的消息暂未测试。

:::
